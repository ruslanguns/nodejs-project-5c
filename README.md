## Introduction

This is a project made with Nodejs and it's framework ExpressJS.
Nothing special just testing purpose.

## Install

- Fork this project
```bash
$ git clone https://github.com/ruslanguns/nodejs-project-5c.git
```

- Install
```bash
$ npm i
```

## Development
Requires a NPM server.

```bash
$ npm run start:dev
```

## Production
Requires a NPM server.

```bash
$ npm start
```
